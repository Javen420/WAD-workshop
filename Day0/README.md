# Day 0 - Web Development Fundamentals

## Quick Start

**Easy Access:** Open [0_directory.html](0_directory.html) in your browser for a clickable interface to all demos and exercises!

## Overview

Day 0 is your introduction to web development fundamentals. This module covers HTML and JavaScript basics through interactive demos and hands-on exercises.

## What You'll Learn

### HTML Fundamentals
- Document structure and common tags
- Semantic elements
- Tables with colspan/rowspan
- Images and links
- Forms and form elements

### JavaScript Fundamentals
- Variables and naming conventions
- Data types and operators
- Arrays and objects
- Conditionals and loops
- Functions and scope

**How to run JavaScript demos:**
1. Open [js-runner.html](js-runner.html) in your browser
2. Click on any JavaScript demo file from the directory
3. Open Developer Tools to see console output:
   - **Windows/Linux:** Press `F12` or `Ctrl + Shift + I`
   - **Mac:** Press `Cmd + Option + I`
   - Or right-click → "Inspect Element" → "Console" tab

## Exercises

### HTML Exercises

### JavaScript Exercises

**Exercise 3 - New Employee Badges** ([ex3.js](Exercises/ex3.js))
- Create variables and print formatted output

- Each builds on previous concepts

**Exercise 10 - Coffee Shop** ([ex10.js](Exercises/ex10.js))

### How to Work on JavaScript Exercises

**Option 1: Using js-runner.html (Recommended)**
1. Open [js-runner.html](js-runner.html) in your browser
2. Click on exercise files from the directory
3. View output in Developer Tools Console
4. Edit files and refresh to see changes

**Option 2: Using Node.js**

```bash
# ... etc
```
